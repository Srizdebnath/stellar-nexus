# Stellar Nexus 🌌
> The Decentralized Compute Marketplace on Soroban.

## 🚀 Overview
Stellar Nexus is a Web3 infrastructure platform that allows developers to deploy serverless logic ("Applets") on the Stellar network. Users can chain these applets together to build automated pipelines without managing servers.

## 🛠 Tech Stack
- **Smart Contracts:** Rust (Soroban SDK)
- **Network:** Stellar Testnet
- **Frontend:** Next.js + Tailwind CSS
- **Wallet:** Freighter

## ⚡ Features
- **Marketplace:** Discover and verify logic applets.
- **Pipeline Builder:** Chain applets (e.g., Text -> Hash -> NFT) efficiently.
- **Low Cost:** Utilizing Soroban's low-fee structure for micro-computations.

## 📦 Usage
1. Connect Freighter Wallet.
2. Browse the Applet Marketplace.
3. Execute logic on-chain.
